# AnnotationAlchemist
Site to convert between image labeling formats

## AAClient

### Getting started:
~~~
cd AAClient
pnpm i
pnpm dev
~~~

## AAApi

### Getting started:
~~~
cd AAApi
conda env create -f environment.yml
python main.py
~~~
